# Kontor

Business-Zeiterfassung (Single-File-PWA). Diese Auslieferung enthaelt nur die leere
App-Huelle. Alle Daten bleiben ausschliesslich im Browser des Geraets: kein Server,
kein Konto, keine Analytics, keine Requests an Dritte.

Quelle und Entwicklung: privates Repo `Kontor`.
